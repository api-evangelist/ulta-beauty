# Ulta Beauty

Ulta Beauty is the largest beauty retailer in the United States, offering cosmetics, fragrance, skin care products, hair care products, and salon services through retail stores and online. The company operates over 1,400 stores across all 50 states and an e-commerce platform.

**URL:** [https://raw.githubusercontent.com/api-evangelist/ulta-beauty/refs/heads/main/apis.yml](https://raw.githubusercontent.com/api-evangelist/ulta-beauty/refs/heads/main/apis.yml)

## Features

| Name | Description |
|------|-------------|
| Online Store | E-commerce platform for cosmetics, fragrance, skin care, and hair care products with loyalty program integration. |
| Salon Services | In-store salon services including hair, skin, and brow services available for booking online. |
| Ulta Beauty Rewards | Loyalty program with points earning on purchases, birthday rewards, and tiered membership benefits. |
| Affiliate Program | Publisher and content creator affiliate program through Impact network with commission on qualifying sales. |
| Beauty Services | Complimentary makeup services, skin consultations, and try-before-you-buy experiences in stores. |
